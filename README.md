Ansi color tool for Rust

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
palett = ">=0.0.1"
```

and this to your crate root:

```rust
extern crate palett;
```

## Compatibility

The `palett` crate is tested for rustc 1.48 and greater.
